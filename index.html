Validación de Datos de Usuaios

La validación de datos es muy importante en un formulario, pues nos permite hacer que el ususario tome el camino deseado en la ejecución del programa.
Los ejemplos que se muestran acontinuación han sido adaptados al lenguaje de javascript, pero cumplen la misma funcionalidad que los programas realizados en visual basic, se representa una forma gráfica que asemeja la ejecución de estos pero no se representa exactamente como en el programa real.

Validación por Control · ComboBox

El control ComboBox muetra una lista de opciones, permitiendo así que el usuario no ingrese datos erroneos.
En el ejemplo se puede observar como al seleccionar un idioma en el ComboBox, el Label de abajo cambia su texto a un saludo en el idioma en cuestión.


Código

*En el código original, el combobox cambia el texto de los label de alrededor, los cuales se
sustrejeron en este ejemplo, se ingresa un nombre en un textbox que se muestra con el saludo, el cual se
muestra al dar click en un boton, como se puede onservar en el ejemplo solo se muestra lo principal.

Private Sub cbxidioma_SelectedIndexChanged(sender As Object, e As EventArgs) Handles cbxidioma.SelectedIndexChanged
'Según la selección se escribira el el label un saludo del idioma, seguido del nombre ingresado.
If cbxidioma.Text = "Español" Then
MsgBox("Hola "& mxtnombre.Text)
ElseIf cbxidioma.Text = "Ingles"Then
MsgBox("Hello "& mxtnombre.Text)
ElseIf cbxidioma.Text = "Frances"Then
MsgBox("Salut "& mxtnombre.Text)
ElseIf cbxidioma.Text = "Italiano"Then
MsgBox("Ciao " & mxtnombre.Text)
End If
End Sub

Validación por Propiedad · Enable

La propiedad Enabled permite habilitar o deshabilitar el uso un control mediante los valores true y false.
En el siguiente ejemplo de un formulario se puede onservar que una vez que se da click al botón guardar, los elementos se deshabilitan, evitando que se interactue con ellos.

Código

*En esta ocasión el ejercicio muestra claramente el funcionamiento que tiene el programa original.

Private Sub btnGuardar_Click(sender As Object, e As EventArgs) Handles btnGuardar.Click
'Al dar click en el boton Guardar la progressbar avanza hasta su máximo y los demás controles exceptuando el boton btnReset se deshabilitan.
pgbCarga.Increment(100)
txtNombre.Enabled = False
txtApellido.Enabled = False
txtUsuario.Enabled = False
nudEdad.Enabled = False
btnGuardar.Enabled = False
End Sub

Private Sub btnReset_Click(sender As Object, e As EventArgs) Handles btnReset.Click
'Los valores de todos los controles pasaran a sus iniciales.
txtNombre.Text = ""
txtApellido.Text = ""
txtUsuario.Text = ""
nudEdad.Text = 0
'ProgressBar tendra un valor de 0 nuevamente, y se activaran los demás controles.
pgbCarga.Value = 0
txtNombre.Enabled = True
txtApellido.Enabled = True
txtUsuario.Enabled = True
nudEdad.Enabled = True
btnGuardar.Enabled = True
End Sub

Validación por Evento · TextChanged

El evento TextChanged ejecuta una acción cuando el texto cambia.
Como se observa en el ejercicio si yo ingreso una contraseña y luego en el espacio de confirmar escribo otra contraseña se muestra un mensaje "Contraseña incorrecta" en cambio si escribo la misma contraseña muestra "Contraseña correcta" y habilita el botón continuar.

Código

*En esta ocasión el ejercicio muestra casi en su totalidad el funcionamiento que tiene el programa original.

Private Sub txtConfirmar_TextChanged(sender As Object, e As EventArgs) Handles txtConfirmar.TextChanged
'Se oculta el texto como contraseña.
txtConfirmar.UseSystemPasswordChar = True
'Cuando el contenido de txtPass sea igual a txtConfirmar por el evento TextChanged se mostrará el mensaje en Label2 y Button3 se habilitara.
'Si no son iguales Button3 se deshabilita.
If txtPass.Text = txtConfirmar.Text Then
Label2.Text = "Contraseña correcta"
Button3.Enabled = True
Else
Label2.Text = "Contraseña incorrecta"
Button3.Enabled = False
End If
End Sub

Private Sub txtPass_TextChanged(sender As Object, e As EventArgs) Handles txtPass.TextChanged
txtPass.UseSystemPasswordChar = True
End Sub
